# Mini-Projet TMA — Stéganographie par LSB

## 📌 Objectif
Ce projet démontre la technique de stéganographie par bits de poids faibles (LSB),
permettant de cacher un message secret dans une image sans altérer son apparence.

---

## ⚙️ Fonctionnalités
✔ Chargement d’une image  
✔ Encodage d’un message secret  
✔ Génération d’une image stégo  
✔ Extraction du message caché  

---

## 🧪 Résultat

Message caché :

L’image stégo est visuellement identique à l’image originale.

---

## 📂 Fichiers
- `Steganographie_SupNum.ipynb` → Notebook principal
- `original_image.png` → Image utilisée
- `stego_image.png` → Image contenant le message caché

---

## 🚀 Comment exécuter
1. Ouvrir le notebook dans Google Colab
2. Exécuter toutes les cellules
3. Observer le message extrait
